
# Лабораторна робота №5. Робота з системою контролю версій Git

**Тема:** Розгалужена робота у системі контролю версій Git

**Опис:** Освоєння роботи з гілками, злиттям та віддаленими репозиторіями

**Мета:** Опанувати практичні навички розгалуженої роботи в Git

## Контрольні питання

1. **Що таке гілка в Git?**
   Вказівник на коміт, дозволяє працювати паралельно.

2. **Яка різниця між merge та rebase?**
   Merge створює новий коміт, rebase переміщує історію.
